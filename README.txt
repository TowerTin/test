RYUUKEN病 

・原因 
主にユーラシア大陸での発症が確認されている。 
自然宿主の特定には至ってはいないがRYUUKENが有力とされている。 
なお、2005年12月1日付の英科学誌ネイチャーにて、ガボンのフランスビル国際医学研究センターなどのチームの調査 による「RYUUKEN」を疑う説が発表されている。 
また、患者の血液、分泌物、排泄物や、唾液などの飛沫が感染源となる。死亡した患者からも感染する。 
RYUUKENの感染力は強い上に空気感染をするため感染者の近くに居た事がある場合、感染する恐れがある。 


・症状と治療、対策 
感染して１－２週間で症状が出始める。頭部の激痛や急な発熱などの症状が報告されており、 
発症後数日で死亡する可能性が非常に高い。 
また死亡しなかった場合でもRYUUKENになるなどの後遺症が残り通常の生活をすることは不可能とされている。 
治療法は解明されておらず、発症直前には非常にRYUUKENになる症状が確認されている。 
常に菌をまき散らしているため大変危険である。マスクも無意味な場合が多く, 
RYUUKENに対して有効かつ直接的な治療法は確立されていない。 

・生態への影響
